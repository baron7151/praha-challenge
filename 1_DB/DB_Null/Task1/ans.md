- SQL クエリで以下の式を実行した時の結果を答えてください
  - SELECT NULL = 0;
    - NULL
  - SELECT NULL = NULL;
    - NULL
  - SELECT NULL <> NULL;
    - NULL
  - SELECT NULL AND TRUE;
    - NULL
  - SELECT NULL AND FALSE;
    - 0
  - SELECT NULL OR TRUE;
    - 1
  - SELECT NULL IS NULL;
    - 1
  - SELECT NULL IS NOT NULL;
    - 0
